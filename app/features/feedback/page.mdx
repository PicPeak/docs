# Feedback

PicPeak allows gallery guests to interact with photos through likes, ratings, and comments.

## Features

### Likes

Guests can like individual photos. Like counts are visible to other guests and tracked in the admin analytics dashboard.

### Ratings

Photos can receive star ratings from guests. Aggregated ratings help photographers identify the most popular shots.

### Comments

Guests can leave comments on individual photos. Comments are visible to other guests and can be moderated by administrators.

## Admin moderation

All feedback is accessible from the admin panel:

- View all comments, likes, and ratings per event.
- Delete inappropriate comments.
- Filter photos by feedback metrics (most liked, highest rated, most commented).

## Filtering and export

Photographers can filter gallery contents by feedback data:

- Sort by number of likes, average rating, or comment count.
- Export filtered selections as ZIP.
- Generate Capture One or Lightroom-compatible file lists for professional post-processing workflows.

## Configuration

Feedback features can be enabled or disabled per event from the event settings in the admin panel.
