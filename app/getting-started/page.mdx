# Getting Started

Get PicPeak running and create your first gallery in three steps:

1. **[Quick Start](/getting-started/quick-start)** — clone the repo, configure `.env`, start Docker. Takes about five minutes.
2. **[First Login](/getting-started/first-login)** — find your admin password and complete the mandatory password change.
3. **[Your First Gallery](/getting-started/first-gallery)** — create a draft event, upload photos, and publish it to your client.
