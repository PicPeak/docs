# Getting Started

This section walks you through getting PicPeak up and running, from initial installation to your first gallery.

<img src="/images/screenshot-dashboard.png" alt="PicPeak Admin Dashboard" style={{ borderRadius: '8px', marginTop: '1rem', marginBottom: '1.5rem', boxShadow: '0 2px 12px rgba(0,0,0,0.1)' }} />

## What you will need

- **Docker** v20.10.0+ and **Docker Compose** v2.0.0+ (recommended path)
- Alternatively, **Node.js** 18+ for building from source
- At least **2 CPU cores**, **2 GB RAM**, and **20 GB storage** (plus space for photos)
- A domain name and SMTP credentials for production use

## Steps

1. **[Installation](/getting-started/installation)** --- pull the Docker images or build from source.
2. **[Quick Start](/getting-started/quick-start)** --- clone, configure `.env`, and start the containers.
3. **[First Login](/getting-started/first-login)** --- find the auto-generated admin password and complete the mandatory password change.
4. **[Configuration](/getting-started/configuration)** --- set URLs, email, and other essential environment variables.
