# Code of Conduct

This repository — the source for [docs.picpeak.app](https://docs.picpeak.app) — follows the same Code of Conduct as the main PicPeak project.

See [`PicPeak/picpeak/CODE_OF_CONDUCT.md`](https://github.com/PicPeak/picpeak/blob/main/CODE_OF_CONDUCT.md) for the full text.

## In brief

- Be respectful. Assume good faith.
- Focus on the work, not the person.
- Disagreements are fine; personal attacks are not.
- Harassment, discrimination, or trolling will result in removal from the project.

## Reporting

Conduct issues can be reported to the maintainer privately at the email listed on the [maintainer's GitHub profile](https://github.com/the-luap).

The same channel handles concerns about both the application repo and this docs repo.
